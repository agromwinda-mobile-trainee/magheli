# Dossier pour l'image du splash screen

Placez votre image `splash.png` dans ce dossier.

## Spécifications de l'image :

- **Format** : PNG
- **Dimensions recommandées** : 1080x1920 pixels (portrait)
- **Fond** : Blanc (#FFFFFF)
- **Contenu** : 
  - Icône du restaurant au centre
  - Texte "MAGHALI" en majuscules
  - Sous-titre "Gestion interne" (optionnel)

Une fois l'image placée ici, exécutez :
```bash
flutter pub run flutter_native_splash:create
```
